# TypeScript for Playwright

This repository is a day-by-day TypeScript practice project.

## What this project is about

- Learning core TypeScript syntax and concepts through small examples.
- Building a strong TypeScript foundation for Playwright automation work.
- Keeping notes and practice files organized by day.

## Project structure

- `Day1` to `Day7`: topic-based TypeScript practice files.
- `.vscode/launch.json`: local debug configuration.

## Topics covered (so far)

- Variables and data types
- Operators
- Loops (`for`, `while`, `do...while`)
- Named and anonymous functions

## Goal

Use this repository as a personal TypeScript reference while learning Playwright test automation.
